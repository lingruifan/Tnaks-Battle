Project Description: Tanks BattleThis project is an 2D tanks shooting game with both single-player, which enemy is an AI, and multi-player modes. In the game, each player will control a tank in a maze and shoot at each other. The only player who stands until the last is the winner. How to run?The game needs to be run on the terminal. For the AI mode, the user needs to run the game file on the terminal and choose the AI Mode on the menu. For the multiplayer mode, other players will be shared with the game directory (zip file through either email or airdrop). Then players will run the game file in the directory and choose the mode of 2/3/4 players to be connected together. LibrariesThere’s no library used in this project.Short-cut keysThere are no short-cut keys used in this project.
